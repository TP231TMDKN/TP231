# Liste Simplement Chainee Trie

## Description
Ce programme implemente une liste simplement chainee triee en langage C. Les elements sont inseres dans la liste de maniere a maintenir un ordre croissant a tout moment.

## Structure de donnees

### Noeud (Node)
Chaque noeud contient:
- data: la valeur entiere stockee dans le noeud
- next: pointeur vers le noeud suivant

## Fonctionnalites

### Fonctions principales

1. **Creation de noeud**: createNode() - Cree un nouveau noeud avec allocation memoire
2. **Insertion triee**: inserer_tri() - Insere un element dans la liste en maintenant l'ordre croissant
3. **Affichage**: afficher_liste() - Affiche la liste du debut a la fin
4. **Menu interactif**: menu_main() - Interface utilisateur pour gerer la liste

### Menu interactif
Le programme offre un menu avec les options suivantes:
1. Inserer dans la liste simple triee
2. Afficher la liste simple
0. Quitter le programme

## Compilation et execution

Pour compiler le programme:
gcc liste_simplement_chaine_trie.c -o liste_triee

Pour executer:
./liste_triee

## Caracteristiques techniques

- Insertion automatique en position triee
- Gestion memoire avec malloc
- Affichage clair avec fleches indiquant le chainage
- Liste se termine par NULL

## Algorithme d'insertion

L'insertion triee fonctionne ainsi:
- Si la liste est vide ou la valeur est plus petite que la tete, insertion en tete
- Sinon, parcourt la liste jusqu'a trouver la position appropriee
- Insere le nouveau noeud a la position trouvee

## Notes d'utilisation

- La liste reste toujours triee
- Les doublons sont autorises
- L'ordre est croissant
- La liberation memoire n'est pas implementee dans ce code
